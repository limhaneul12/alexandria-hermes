# Chat History

The plugin keeps the latest 20 interactions in the pane for the current Obsidian
session. Entries can be clicked to restore the answer, workflow state, delegate
panel, sources, and graph actions.

This history is intentionally local UI state; it is not written to Obsidian
unless the user approves transcript saving or creates a note.
