# Obsidian Librarian Chat Window

## Goal

Upgrade the Obsidian side pane from a single answer renderer into a compact
conversation and workflow console for Alexandria-Hermes.

## Delivery

- Session-local conversation history.
- Workflow and delegate status badges.
- Approval action cards for LangGraph resume.
- Separate GPT OAuth librarian result panel.
- Save-as affordances for context, skill draft, and prompt template notes.
