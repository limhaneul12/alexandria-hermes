# Save Affordances and Safety

The pane can now save the current answer as:

- context note;
- skill draft;
- prompt template.

OAuth tokens and provider secrets remain backend-only. The plugin only stores the
optional operator key in Obsidian plugin settings and does not write OAuth token
material into Markdown notes.
