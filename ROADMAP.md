# Alexandria-Hermes Roadmap

## Current status

Alexandria-Hermes is an active MVP for local-first, single-operator agent library and recall workflows.

Verified local surfaces include:

- skill/prompt/library management
- thin candidate search and selected full-load paths
- Context Vault save/lint/recall/chunks/archive/access events
- RAG health and FTS/vector strategy controls
- durable Memory Compact pages/API/CLI/MCP tools
- librarian brief/packet and optional provider delegation surfaces
- Hermes onboarding, policy, and MCP server integration
- Next.js UI for dashboard, library, contexts, memory compacts, RAG inspector, librarian chat, and settings
